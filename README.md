# Wedding Invitation Theme

This package is a static website template based on the provided Tomi & Airys invitation.

Quick edit guide:

1. Open `invitation-data.js`.
2. Update names, date, venues, RSVP link, entourage, and timeline.
3. Replace the images inside the `images` folder if you want to use this for another couple.
4. Open `index.html` in a browser or upload the folder to your hosting service.

Files:

- `index.html` — page structure
- `styles.css` — theme styling
- `app.js` — theme interaction and dynamic rendering
- `invitation-data.js` — editable invitation content
